Thank you for downloading Mission Key Collector!
This extension automatically claims your daily mission key from the Star Trek Fleet Command web store.
It also includes a manual "Claim Now" button in the extention popup.

Because this is an unpacked extension (not from the Chrome Web Store), you need to enable Developer mode in Chrome to install it.

STEP-BY-STEP INSTALLATION INSTRUCTIONS
──────────────────────────────────────

1. Unzip the downloaded file
   - Extract all files from the ZIP to a folder on your computer.
     (Example: C:\Extensions\Mission-Key-Collector or Desktop\Mission Key Collector)

   You should see files like:
   - manifest.json
   - popup.html
   - popup.js
   - content.js
   - background.js
   - icons/ (with icon-16.png, icon-48.png, icon-128.png)
   - README or this text file

2. Open Chrome and go to the Extensions page
   - In Chrome, type this in the address bar and press Enter:
     chrome://extensions/

3. Enable Developer mode
   - In the top-right corner of the chrome://extensions/ page, look for the switch labeled
     "Developer mode"
   - Toggle it ON (it turns blue).

4. Load the extension
   - A new button appears near the top-left: "Load unpacked"
   - Click "Load unpacked"
   - Navigate to the folder where you extracted the files (the folder that contains manifest.json)
   - Select that folder and click "Select Folder" (or "Open")

5. The extension should now appear in the list
   - You will see "Mission Key Collector" with its icon (Starfleet delta with Enterprise).
   - If the icon still shows as a gray puzzle piece in the toolbar, don't worry — that's normal for unpacked extensions.
     Pin it anyway (see step 6) — it will work perfectly even if the toolbar icon is the puzzle.

6. Pin the extension to your toolbar (optional but recommended)
   - Click the puzzle piece icon in the top-right of Chrome (Extensions button).
   - Find "Mission Key Collector" in the list.
   - Click the pin icon (📌) next to it — it turns blue.
   - The extension icon (or puzzle if in dev mode) now stays visible in your toolbar.

7. Test it
   - Click the extension icon (or puzzle if unpinned) → the popup should open.
   - You can click "Claim Now" manually, or wait for the automatic daily claim at ~00:01 server time.
   - Open https://home.startrekfleetcommand.com/store in a tab — the extension will auto-run when needed.

8. For Auto Claim: You must have you browser open, and already be logged into the scopely store. THIS WILL NOT LOG YOU IN AND DOES NOT STORE YOUR LOGIN DATA.

That's it! You're ready to collect those daily mission keys hands-free.

Troubleshooting
───────────────
- No icon in toolbar? → Normal in developer mode. Pinning still works; functionality is unaffected.
- Extension not loading? → Make sure you selected the folder containing manifest.json (not a subfolder).
- Errors on chrome://extensions/? → Check the "Errors" section on the card and make sure all files are present.
- Want to update later? → Just replace files in the folder → click the circular Reload arrow on the extension card.
